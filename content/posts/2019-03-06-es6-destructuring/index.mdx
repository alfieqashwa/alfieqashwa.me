Destructuring pada dasarnya hanya meringkas cara menulis `code`.

### `Array Destructuring`

Perhatikan perbedaan cara penulisan sebelum JavaScript edisi `ES6`:

```js
// ES5

var array = [1, 2, 3]
var first = array[0]
var third = array[2]

console.log(first, third) // 1 3
```

Menggunakan `Array destructuring`:

```javascript {3}
// ES6: array destructuring
const array = [1, 2, 3]
const [first, second, third] = array

console.log(first, second, third) // 1 2 3
```

dan jika tidak menyertakan `the second index`, gunakan `white-space`:

```js {3}
// ES6: array destructuring
const array = [1, 2, 3]
const [first, , third] = array // add white-space at the second index

console.log(first, third) // 1 3
```
