## `Definisi sebuah Function`

_Functions_ atau fungsi didefinisikan atau dideklarasikan oleh _keyword_ `function`.

```js
function namaFungsi() {
  // code...
}
```

Deklarasi di awali dengan _keyword_ `function` diikuti dengan nama fungsi. Nama fungsi ketentuannya sama seperti _variables_ -- bisa berbentuk karakter, angka, _underscores_ `_` dan _dollar signs_ `$`, dan umumnya ditulis dengan [camel case](https://en.wikipedia.org/wiki/Camel_case). Parameter terletak di dalam tanda kurung atau _parentheses_ dimana sifatnya opsional -- tidak mesti ada. _Code_ dituliskan di dalam _block-scope_ di antara tanda kurung kurawal atau _curly brackets_.

```js
function salam() {
  console.log("Assalaamu 'alaikum, Gaesss!")
}
```

Untuk menjalankan fungsi di atas, _kudu_ dipanggil atau bahasa teknisnya _mesti_ di _invoke_. Cara panggilnya dengan menuliskan nama _variable_ diikuti dengan _parentheses_.

```js {5}
function salam() {
  console.log("Assalaamu 'alaikum, Gaesss!")
}

salam()
```

```terminal
ğŸ”´ OUTPUT:
Assalaamu 'alaikum, Gaesss!
```

Dengan fungsi `salam()` kita dapat _reuse_ atau menggunakan kembali berapakalipun yang diperlukan. Menggunakan _parameter_, _code_ akan menjadi dinamis.

## `Returning Values`

Kita kembali ke `Kisah Papan Tulis` yang dibahas di awal. Waktu SD kita diajarkan matematika dasar menghitung luas permukaan dengan rumus. Rumus itulah yang kita aplikasikan ke dalam fungsi.

Daripada menuliskan cara seperti...

```js
let panjang = 240,
  lebar = 120

let papanTulis1 = panjang * lebar
console.log(papanTulis1)

panjang = 120
lebar = 90

papanTulis2 = panjang * lebar
console.log(papanTulis2)
```

...kita akan merepresentasikan `rumus` ke dalam fungsi, dalam contoh ini adalah `rumus luas persegi panjang`.

```js {1-3}
function luasPersegiPanjang(panjang, lebar) {
  return panjang * lebar
}

// invoke function
luasPersegiPanjang(240, 120)
luasPersegiPanjang(120, 90)
```

## `Function Expressions`

Sebelumnya kita mendeklarasikan fungsi untuk mendapatkan luas dari dua angka dan menghasilkan nilai berdasarkan rumus yang kita buat di dalam _block-scope_ -- dalam hal ini adalah `panjang * lebar` yang ditulis setelah _keyword_ `return`.

Kita juga dapat membuat `function expression` ke dalam sebuah `variable`.

```js {1}
const luasPapanTulis = function luasPersegiPanjang(panjang, lebar) {
  return panjang * lebar
}

// invoke function
luasPapanTulis(240, 120)
luasPapanTulis(120, 90)
```

### `Anonymous Function`

Umumnya, suatu fungsi yang di `assign` ke dalam sebuah `variable` ditulis tanpa nama, disebut `Anonymous Function`.

```js {2}
// unnamed function
const boardArea = function(height, width) {
  return (height * width) / 100
}
```

Pada artikel sebelumnya dikupas tentang [Arrow Function](https://www.alfieqashwa.me/arrow-function/) ğŸ‘ˆ _so to muscle our memories_ mari kita konversi _code_ sebelumnya.

```js {2}
// convert into arrow function
const boardArea = (height, width) => {
  return (height * width) / 100
}
```

Atau boleh juga diringkas seperti ini:

```js
const boardArea = (h, w) => (h * w) / 100
```

Atau jika contoh yang hanya membutuhkan satu parameter, misal `rumus bujursangkar`, maka _parentheses_ sifatnya opsional.

```js {1,4}
const square = x => x * x

// invoke
square(5)
```
